 This software equips INSAT \href{https://arxiv.org/abs/2101.12548}{\texttt{ \mbox{[}1\mbox{]}}}, \href{https://arxiv.org/abs/2210.08627}{\texttt{ \mbox{[}2\mbox{]}}} with parallel processing capability, namely \href{https://arxiv.org/abs/2401.08948}{\texttt{ {\itshape PINSAT}}}, using the ideas developed in \href{https://arxiv.org/abs/2203.01369}{\texttt{ e\+PA$\ast$\+SE}}.

This particular fork catkinizes the workspace to be able to build with ROS and implements \href{https://github.com/sbpl/parallel_search/blob/catkin/examples/manipulation/MoveitInterface.hpp}{\texttt{ wrapper functions}} to be used by the \href{https://github.com/sbpl/shield_planner/}{\texttt{ Shield Planner}}\hypertarget{index_autotoc_md1}{}\doxysubsection{Dependencies}\label{index_autotoc_md1}

\begin{DoxyEnumerate}

\item Open\+CV C++ for visualization
